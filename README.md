# Thumb
Thumbnail generation for PHP

__Note: This branch is not considered stable, and is prone to changes__

## Installation
The recommended way to install is through [Composer](https://getcomposer.org/).
```JSON
{
    "require": {
        "xicrow/php-thumb": "dev-develop"
    }
}
```
